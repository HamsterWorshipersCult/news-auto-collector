# News Auto Collector

A WordPress plugin that collects RSS news feeds automatically and displays them using a shortcode.

## Features
- RSS feed aggregation
- Automatic hourly updates via WP-Cron
- Manual update button
- Shortcode display `[news_collector]`

## Installation
1. Upload plugin to `/wp-content/plugins/`
2. Activate in WordPress admin panel
3. Add RSS feeds in admin menu
